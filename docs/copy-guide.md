# 문구 초안

## 브랜드

- 메인: `기온별플리`
- 서브: `By Degrees`

## 입력 단계

### 이름 / 닉네임

- 제목: `이름 또는 닉네임을 적어주세요`
- placeholder: `예: 민지, summerday, byul`
- 설명: `결과 이미지에 함께 표시돼요`

### 아티스트 옵션

- 제목: `어떤 방식으로 만들고 싶나요?`
- 옵션 1: `모두 같은 아티스트의 곡으로 구성하고 싶어요`
- 옵션 2: `다양한 아티스트의 곡으로 구성하고 싶어요`

### 단일 아티스트 입력

- 제목: `아티스트명을 적어주세요`
- placeholder: `예: Red Velvet, NewJeans, Taylor Swift`

## 플레이리스트 생성 화면

- 제목 placeholder: `기온별 플레이리스트`
- 아티스트 placeholder: `아티스트명을 입력하세요`
- 버튼: `이미지 생성하기`
- 곡 추가: `곡 추가`
- 검색 모달 제목: `곡 찾기`
- 검색 입력 placeholder: `아티스트명 또는 곡명으로 검색`

## 결과 / 공유

- 이미지 저장: `이미지 저장`
- X 공유: `X로 공유`
- 보조 문구: `나만의 기온별플리를 저장하고 공유해보세요`

## 해시태그 초안

- 공통: `#기온별플리`
- 단일 아티스트 예시: `#기온별레드벨벳`
- 영문 예시:
  - `#TracksByDegrees`
  - `#RedVelvetByDegrees`

### Note

- `#RedVelvetByDegrees`는 아주 이상하진 않지만, 팬덤 자연어 감각으로는 조금 딱딱할 수 있습니다.
- MVP에서는 한국어 해시태그 중심이 더 자연스럽습니다.

## 크레딧 문구

- 기본 표기: `© 2026 기온별플리 (By Degrees)`

## 관리자

- 제목: `운영 통계`
- 카드:
  - `방문자 수`
  - `이미지 저장 횟수`
  - `이미지 및 링크 공유 횟수`
  - `국가별 방문자 수`
  - `대륙별 방문자 수`
