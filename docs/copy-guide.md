# 문구 초안

## 브랜드

- 메인: `기온별플리`
- 서브: `By Degrees`

## 입력 단계

### 이름 / 닉네임

- 제목: `이름 또는 닉네임을 적어주세요`
- placeholder: `예: 민지, summerday, byul`
- 설명: `결과 이미지에 함께 표시돼요`

### 아티스트 옵션

- 제목: `플레이리스트를 이렇게 구성하고 싶어요`
- 옵션 1: `모두 같은 아티스트의 곡으로`
- 옵션 2: `다양한 아티스트의 곡으로`

### 단일 아티스트 입력

- 제목: `아티스트를 선택해주세요`
- placeholder: `예: Red Velvet, NewJeans, Taylor Swift`

## 플레이리스트 생성 화면

- 제목 규칙: `기온별 {아티스트명} by {닉네임} 🎶`
- 복수 아티스트 제목: `기온별 플리 by {닉네임} 🎶`
- 버튼: `플레이리스트 미리보기 ✨`
- 빈 구간 토스트: `모든 기온 구간에 한 곡\n이상을 선택해주세요.`
- 중복곡 확인: `이미 추가한 곡입니다. 그래도 추가하시겠어요?`
- 곡 추가: `곡 추가`
- 검색 모달 제목: `곡 찾기`
- 검색 입력 placeholder: `아티스트명 또는 곡명으로 검색`

## 결과 / 공유

- 이미지 저장 안내: `내 기온별플리 저장하기`
- 저장 안내 토스트: `👆 위 이미지를 길게\n눌러 저장해주세요.`
- X 공유: `X로 공유`
- 링크 복사: `링크 공유`
- 링크 복사 토스트: `링크가 복사되었습니다.`
- 공유 섹션 제목: `내 기온별플리 공유하기`
- 링크 공유 URL: `https://by-degrees.vercel.app`

## 점검 / 오류 안내

- 음악 검색 오류: `음악 검색 요청이 몰려 잠시 쉬는 중이에요.\n잠시 후 다시 시도해주세요.`
- 점검 팝업 제목: `잠시 점검 중입니다`
- 점검 팝업 본문: `음악 검색 요청이 몰려 잠시 쉬어가고 있어요.\n조금 뒤 다시 찾아와주세요.`
- 점검 완료 알림 버튼: `점검 완료 알림 받기`
- 점검 완료 알림 설명: `점검 완료 시 메일로 안내드려요.`
- 개인정보 안내: `입력한 이메일은 점검 완료 1회 안내 목적으로만 수집합니다. 메일 발송 후 발송 여부 기록과 함께 3일간 보관되며, 이후에는 즉시 삭제됩니다.`

## 해시태그 초안

- 공통: `#기온별플리`
- 단일 아티스트 예시: `#기온별RedVelvet`
- 복수 아티스트: `#기온별플리 #ByDegrees`
- 단일 아티스트: `#기온별플리 #기온별{아티스트명} #{아티스트명}ByDegrees`

- X 공유 텍스트의 제목/본문은 언어별로 바뀌어도 해시태그 정책은 한국어/영어 플로우 모두 동일하게 유지합니다.
